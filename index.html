Droneageddon ⚡

Droneageddon is a simple drone simulator which uses reinforcement learning to train drones to track targets!

Do you want to try flying one yourself?

Use the ← and → keys to drive the left and right motor respectively. Heads up it isn't easy 😂

Don't fancy flying yourself? Try training some drones. Warning this will take sometime!

Want to save some time? Load a already training drone and try it out.
